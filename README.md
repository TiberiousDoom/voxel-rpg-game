1. Clone this repository
2. Run `npm install`
3. Run `npm start`
4. Open [http://localhost:3000](http://localhost:3000)

## Controls

- **WASD** - Move
- **Mouse** - Aim
- **Left Click** - Melee attack
- **1-5** - Cast spells
- **E** - Interact with dungeons/base
- **I** - Open inventory
